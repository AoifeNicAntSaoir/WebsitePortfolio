Automated CI/CD

Project Description

During my work placement, I implemented an automated Continuous Integration and Continuous Deployment Pipeline using Teamcity and Octopus Deploy. Every time a change was committed to Github, a build kicked off in Teamcity. The Teamcity built the .sln file, ran NUnit tests, created and deployed a release to Octopus Deploy, pushed a release, deployed a release to an IIS server in a Staging Environments IIS Server (AWS VM) and the promoted the release to the Production Environment IIS Server (Azure VM)
